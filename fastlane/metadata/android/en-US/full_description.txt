Companion app for the Lyrion Custom Data web dashboard, a self-hosted
now-playing view for Lyrion Music Server (formerly Logitech Media
Server / Squeezebox).

Features:

* Now playing: cover art, title, artist and album of the active player,
  refreshed live, with an accent colour derived from the cover
* Synced lyrics displayed karaoke-style, with manual scrolling and a
  resume-auto-scroll control
* Library statistics (albums, artists, plays, genres, ...)
* Automatic discovery of the Lyrion server on the local network
* Keeps the screen on while the dashboard is displayed (configurable)

This app requires a running Lyrion Custom Data server instance on your
network — see the project page for server setup.
